# Vastra Saree House

A complete e-commerce saree website with:

- Home page
- Product listing and detail modal flow
- Shopping cart
- Checkout/payment page
- Login page
- Master admin dashboard
- Backend APIs with JSON persistence

## Run

```powershell
node server.js
```

Then open:

`http://localhost:3000`

## Demo Logins

- Admin: `admin@vastrasaree.com` / `admin123`
- Customer: `customer@vastrasaree.com` / `user123`

## Project Structure

- `server.js` - Node backend and static file server
- `data/store.json` - Products, users, orders, store settings
- `public/` - Frontend pages, styles, and scripts
